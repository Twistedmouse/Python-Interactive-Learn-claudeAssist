## Testing (1 Minute)

### Test Terminal:
1. Click "👨‍💻 Learn" tab
2. Write code: `print("hello")`
3. Click "▶ Run Code"
4. See "hello" in terminal below ✅

### Test Snippets:
1. Write some code
2. Click "💾 Save Attempt"
3. See "✅ Code saved!"
4. Click "📝 Previous (1)"
5. See your saved code ✅

### Test Certificate:
1. Complete 1 topic ("✓ Mark Complete")
2. Go to "🏆 Achievements"
3. Complete all topics (or skip to test)
4. See certificate section
5. Enter name, click download ✅
