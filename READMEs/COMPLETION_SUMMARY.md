## Progress Tracking

### What Gets Saved
```javascript
{
  "1-0": true,        // Phase 1, Topic 0 (Variables) completed
  "1-1": true,        // Phase 1, Topic 1 (Lists) completed
  "2-3": false,       // Phase 2, Topic 3 (APIs) in progress
  "3-1": true,        // Phase 3, Topic 1 (Prompt Engineering) completed
}
```

### How It Works
1. **Mark Complete** button saves to localStorage
2. **Next time you visit** → progress is restored
3. **Progress bar** shows overall completion percentage
4. **Can resume** any topic by clicking it in the Roadmap

### Storage Location
- **Browser localStorage key**: `learning_roadmap_progress`
- **Data persists** across sessions (unless you clear browser data)
- **Safe for**: Chromebooks, iPad, Windows, Mac, Linux

## Technical Details

### What Gets Persisted
- Topic completion status (true/false per topic)
- Nothing else (view state, code, etc. is temporary)

### Limitations
- **Doesn't sync across devices** (stored locally)
- **Clears if** you clear browser cache/storage
- **One account per browser** (no user accounts)
